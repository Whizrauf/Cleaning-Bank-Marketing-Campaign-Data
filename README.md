# Bank Marketing Campaign Data Cleaning

## Project Overview

Data cleaning is an essential skill for data engineers, involving tasks such as reading, modifying, splitting, and storing data. In this project, you will apply your data-cleaning skills to process and analyze information about marketing campaigns run by a bank.

## Objective

The main goal of this project is to clean and transform the bank marketing campaign data, ensuring it is in a usable format for further analysis or modeling. This will include tasks such as:

- Modifying column values
- Adding new features
- Converting data types
- Handling missing values
- Saving the cleaned data into multiple files

## Features

- **Data Cleaning**: Cleaning and transforming raw data to remove inconsistencies.
- **Feature Engineering**: Creating new features from existing data to enhance the analysis.
- **Data Type Conversion**: Converting data types to ensure consistency across columns.
- **Missing Value Handling**: Identifying and addressing missing or incorrect data.
- **File Output**: Saving the processed data into different file formats for further use (e.g., CSV, JSON).
